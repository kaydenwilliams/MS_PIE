# MS Progression Analytics

Processes MS clinical data to track EDSS, relapses, MRI lesions, and symptoms; provides cleaned SQL datasets, Python analysis, forecasting, and interactive Tableau dashboards.

## What it is
A clinical analytics system that processes MS time-series data (visits, symptoms, MRI metrics) to quantify progression patterns, identify relapse trends, and generate objective forecasts.

## What it does
- Ingests raw multi-source MS data
- Cleans, normalizes, and structures it in SQL
- Computes progression velocity, relapse intervals, and lesion-change metrics
- Runs statistical forecasting on EDSS trends and relapse probability
- Outputs dashboards highlighting trajectories, risk patterns, and correlations

## What it includes
- Synthetic clinical datasets
- SQL schema and analytical views
- Python pipeline for extraction, cleaning, analysis, and forecasting
- Tableau dashboards for progression, relapse, and imaging alignment
- Optional lightweight API layer for insight retrieval
